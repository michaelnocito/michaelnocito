# Hi, I'm Michael Nocito

I love learning and the science of how we learn. I'm just as drawn to data and the science behind it, and most of what I build comes from putting those two together. Data analyst in Glassboro, NJ, with eight years as an implementation analyst: migrating enterprise contract data, owning projects end to end, and building processes from scratch as a small company scaled. Day to day I work in SQL, Python, and Excel, turning messy data into a clear answer.

## Analysis & Portfolio

### 📈 [Telco Customer Churn: Exploratory Data Analysis](https://github.com/michaelnocito/telco-churn-analysis)
A hands-on analysis of **why telecom customers leave** and which signals predict churn early enough to act on, across 7,043 customers cleaned and explored in Python/pandas. It walks from raw data through cleaning and feature engineering to plain-English findings on contract type, tenure, charges, and add-on services.

**→ [telco-churn-analysis](https://github.com/michaelnocito/telco-churn-analysis)** &nbsp;·&nbsp; Python · pandas · Google Colab

---

### 📊 [Analyst Prep Kit: Browser-Based Prep for Data Analytics](https://michaelnocito.github.io/analyst-prep-kit/)
A free, browser-based suite covering the tools interviewers actually test: SQL, Excel, Python, Tableau, Power BI, Statistics, and interview prep. Lessons use read-it-aloud code breakdowns, hands-on drills, and a skill readiness score, all in real in-browser environments (live SQLite, Pyodide terminal, Chart.js). No install, no login. Plenty is free to dig into, with an All-Access Pass for the deeper interview tracks, mock exams, the analyst job sim, and the final exam.

**→ [analyst-prep-kit](https://github.com/michaelnocito/analyst-prep-kit)** &nbsp;·&nbsp; JavaScript · sql.js · Pyodide · Chart.js

---

### 🗃️ [Spreadsheet Archaeology: A Data-Analyst Job Sim](https://michaelnocito.github.io/spreadsheet-archaeology/)
A day-in-the-life simulation: you inherit a drive of cursed spreadsheets and learn the real habits of a working analyst by de-cursing them. You read the data, fix type and consistency problems, and leave every file better than you found it. A [Tableau version](https://michaelnocito.github.io/tableau-archaeology/) teaches the same skills for dashboards.

**→ [spreadsheet-archaeology](https://github.com/michaelnocito/spreadsheet-archaeology)** · **[tableau-archaeology](https://github.com/michaelnocito/tableau-archaeology)** &nbsp;·&nbsp; JavaScript

---

### 🧮 [Excel Interview: A Live, In-Browser Excel Assessment](https://michaelnocito.github.io/excel-interview/)
A hiring tool that lets a manager watch a candidate do real Excel work on a video call instead of a cheatable take-home. Real grid plus formula engine, automatic grading that tells a computed formula from a hardcoded value, and a sealed, tamper-evident result.

**→ [excel-interview](https://github.com/michaelnocito/excel-interview)** &nbsp;·&nbsp; React · Univer · client-side

## Tools & Learning Games

- **[SQL Quest](https://michaelnocito.github.io/sql-quest/)**: a tower-defense game where real SQL queries control your turret, 23 waves from `SELECT` to `GROUP BY`. Live SQLite in the browser. → [sql-quest](https://github.com/michaelnocito/sql-quest)
- **[NEXUS](https://michaelnocito.github.io/nexus-sql-mystery/)**: a desktop game where you uncover corporate fraud by writing real SQL against a live database. → [nexus-sql-mystery](https://github.com/michaelnocito/nexus-sql-mystery)
- **[RecordForge](https://michaelnocito.github.io/recordforge/)**: a Windows app plus Python/pip package that generates realistic fictional test files (PDF, Word, Excel, HTML) for QA, demos, and ETL testing. → [recordforge](https://github.com/michaelnocito/recordforge)
- **[Spreadsheet Cleaner](https://michaelnocito.github.io/spreadsheet-cleaner/)**: a guided beginner Python project where you build a working spreadsheet auditing and cleaning tool, layer by layer. → [spreadsheet-cleaner](https://github.com/michaelnocito/spreadsheet-cleaner)
