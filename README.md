# Hey, I'm Michael Nocito 👋

I'm a data analyst based in Glassboro, NJ — transitioning from 8 years 
as an implementation analyst into full-time data work. I spent nearly a 
decade migrating contract data between enterprise systems, managing 
projects from kickoff to training, and building processes from scratch 
as a small company scaled. Now I'm channeling all of that into Python, 
SQL, and Tableau — and building tools I actually want to exist.

---

## What I'm Working On

### 🧼 Spreadsheet Cleaner
A Python learning project built in 3 layers — Basic, Intermediate, and 
Advanced. Students build a real data-cleaning tool, the kind used in 
professional data migration work. No toy examples.

👉 [Check it out](https://github.com/michaelnocito/spreadsheet-cleaner)

### 🌱 Biodiversity Tracker *(coming soon)*
Using a webcam, Python, and data analytics to monitor plant growth and 
biodiversity over time — starting with my own terrarium and scaling up. 
The goal: use data to help people make their little corner of the world 
greener.
